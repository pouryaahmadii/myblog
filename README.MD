# myblog

یک پروژه وبلاگ ساده با Django برای مدیریت پست‌ها و نمایش محتوا.

## ویژگی‌ها
- مدیریت پست‌ها با پنل ادمین Django
- صفحه اصلی با لیست پست‌ها و صفحه جزئیات هر پست
- پیاده‌سازی سیستم دسته‌بندی و برچسب‌ها
- آماده برای توسعه و اضافه کردن قابلیت‌های بیشتر

## پیش‌نیازها
- Python 3.11+
- Django 5+
- pip
